Research and Projects

  • Dynamic Context Detection from Spoken Text and Voice Command in Home Automation System
    Description: Identifying contexts or meanings from voice commands and conversations with smart assistants can contribute to the autonomous control of smart home devices and appliances. To improve automation, there is a growing need for efficient context detection that eliminates the need to memorize voice commands. To address this need, we followed a two-step approach in our research. In the first step, we developed a unique context recognition model using a transformer, an attention mechanism, and a fully connected neural network. We trained this model on a conversational dataset of daily interactions, which demonstrated higher accuracy than some baseline models. In the second step, we developed a dynamic context detection framework that groups new conversational data to automatically identify new contexts. We then merge these newly identified contexts with the initial context recognition model, which helps the model learn new contexts and adapt to a changing environment. Preliminary results show that this two-step approach significantly enhances the adaptability of smart home systems. It enables dynamic context detection and improves the quality of human-computer interactions.
  • Context Recognition from Voice Commands in Smart Home
    Description: To perform context recognition from short voice conversations, we can convert audio to text and apply natural language processing (NLP) techniques to analyze the textual content. However, the existing text classification systems that apply NLP techniques to extract meaningful information from texts require large training data. In this paper, we propose a novel framework to extract contexts from short-spoken texts requiring smaller training datasets. This framework exploits the power of transfer learning and uses a fully connected neural network aided with SBERT encoding, and an attention mechanism. Our proposed framework has been evaluated using two datasets containing short smart home commands. Evaluation results demonstrate that our model achieves higher accuracy in context recognition with low computational costs and less training time compared to other methods like BERT and deep neural networks.
  • Application of Metamorphic Testing for Verification of Neural Network Model in Home Automation System
    Description: The aim of this project is to apply the metamorphic testing technique to validate a neural network model for context recognition from spoken texts for home automation. The proposed approach involves the systematic design and application of metamorphic relations, generate test cases based on those relations, and assess the model's ability to withstand the variations in the input data. Metamorphic relation is basically the relation of how the changes in the input can affect the changes in output. In this project, we propose 10 metamorphic relations to perform the test and identify the faults in our context recognition model. The experimental results show that metamorphic testing can successfully find the defects in our context recognition model. Thus, it opens more scopes for the improvement of the model and enhancing trustworthiness.
  • Students Attention Monitoring System using Facial Landmark Points
    Description: In this project, we developed a system that combines transfer learning with deep neural network techniques to measure attention levels. The project utilizes the DAiSEE dataset, consisting of video snippets capturing user activities such as boredom, confusion, engagement, and frustration. Preprocessing involves extracting frames from the videos and applying MediaPipe's FaceMesh detection to obtain facial mesh images. The key feature of this system is Transfer learning which exploits the use of Xception architecture for image classification. The trained model is evaluated on a validation split from the dataset, yielding an accuracy of 65.3% for multi-label classification. We explored another baseline model of a fully connected neural network (FCN) that yields an accuracy of 63.3% over two classes: engaged and not engaged. The project provides insights into monitoring student engagement and proposes future work to address dataset imbalance and explore binary classification approaches.
  • Research Paper Classification: A Neural Network Approach with Feature Selection and TF-IDF
    Description: With the rapid advancement of computer and information technology, traditional categories for research papers often fail to reflect emerging multidisciplinary fields, making it challenging for users to find relevant research. To address this, we propose a research paper classification system based on metadata, particularly titles and abstracts, due to their public availability. Using feature selection and TF-IDF vectors, we trained a neural network classifier, achieving an accuracy of 77.78% and a precision of 81%. These results highlight the potential for further improvements by exploring advanced classification methods and text representation techniques.
  • Text Analysis: An end-to-end general-purpose easy-to-use thematic and sentiment analysis tool
    Description: Given the sheer volume and complexity of textual data that is available as well as collected on a daily basis in today’s world, it has become an incredibly difficult task to extract useful information and uncover meaningful data from them. Apart from limited ability to process large amount of texts manually by humans, we also introduce our own biases in the analysis process which clouds our judgement and produces skewed results. With recent major advancements in Machine Learning (ML) and Natural Language Processing (NLP), many techniques have been established to perform sentiment and thematic analysis from texts that can perform analysis on large amount of textual data identifying key components for deriving actionable insights. However, these techniques are intricate to understand and implement, often case-specific, and require advanced expert knowledge, which makes it impossible for general users to get benefit from them. In this project, we present a text analysis tool that is easy to use and performs effective thematic and sentiment analysis on a wide variety of texts without requiring to code and knowledge of NLP/ML as it offers higher level of abstraction and versatility.
